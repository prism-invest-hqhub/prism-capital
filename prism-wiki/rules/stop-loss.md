# 止损规则

亏损的底线，不可商量。

---

## 硬性红线

| 规则 | 条件 | 动作 | 例外 |
|------|------|------|------|
| 单笔止损 | 亏损 > 8% | 无条件卖出 | 无 |
| 总账户止损 | 总亏损 > 300元 | 暂停操作一周 | 无 |
| 强赎回避 | 已发强赎公告 | 禁止买入 | 无 |

## 为什么是8%？

- 1万本金 × 8% = 800元 → 单次最大可承受亏损
- 配合3000元单赛道上限，8% = 240元实际单笔最大亏损
- 超过8%说明判断错误，继续持有是赌不是投

## 为什么是300元总止损？

- 1万本金 × 3% = 300元 → 总账户安全线
- 触达说明策略或环境有问题
- 暂停一周 = 冷静期，防止报复性交易

## 止损执行要点

1. **不犹豫**：到线就卖，不看"会不会反弹"
2. **不调整**：8%是8%，不是"8.5%也行"
3. **不例外**：任何理由都不豁免
4. **记入日志**：每次止损记录原因，供复盘
