# 索罗斯反身性

认知影响现实，现实反馈认知——反身循环创造拐点机会。

---

## 核心理论

传统经济学假设：市场参与者基于完整信息做理性决策
索罗斯认为：参与者的认知本身就会改变现实

```
认知偏差 → 错误定价 → 市场行为 → 影响基本面 → 强化偏差
     ↑                                              │
     └──────────────── 反身循环 ─────────────────────┘
```

## 两个函数

- **认知函数**：参与者试图理解世界 → 形成预期
- **参与函数**：参与者的行为改变世界 → 影响基本面

当两个函数同时作用，就形成反身性循环。

## 反身性在可转债中的体现

| 场景 | 反身机制 |
|------|---------|
| 正股下跌→下修预期 | 股价跌→市场预期下修→博弈资金入场→转债止跌 |
| 强赎博弈 | 股价涨→触发强赎→持债人转股→卖压增大→股价回落 |
| 抢权配售 | 发债公告→正股被抢→配售后抛压→股价回落 |

## 拐点识别

反身性循环不可能永远持续，拐点出现条件：

1. **认知与现实差距过大**：泡沫无法维持
2. **外部冲击**：政策变化、黑天鹅事件
3. **资金耗尽**：接盘侠不够了

## 棱镜应用

- 索罗斯是第四层：**市场错在哪？**
- 当反身性制造极端，逆向入场
- 关键不是预测拐点精确时间，而是识别拐点方向

---

> 反身性告诉你市场什么时候在犯错。但犯错可以持续很久——所以需要安全边际保底。
