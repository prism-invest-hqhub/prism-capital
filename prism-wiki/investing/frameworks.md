# 投资框架总览

棱镜的五层投资框架，从价值锚定到量化执行，层层叠加。

---

## 五层框架

```
┌─────────────────────────────┐
│  第五层：西蒙斯量化执行       │  数据驱动，概率优势
├─────────────────────────────┤
│  第四层：索罗斯反身性判断     │  认知偏差→市场扭曲→拐点捕捉
├─────────────────────────────┤
│  第三层：马克斯周期定位       │  钟摆位置判断，逆向布局
├─────────────────────────────┤
│  第二层：芒格多元思维         │  跨学科格栅，逆向排除
├─────────────────────────────┤
│  第一层：格雷厄姆安全边际     │  价格 < 价值的2/3，底座
└─────────────────────────────┘
```

## 框架间关系

| 框架 | 核心问题 | 输出 |
|------|---------|------|
| 格雷厄姆 | 值不值得买？ | 安全边际量化 |
| 芒格 | 会不会死？ | 排除清单（不碰什么） |
| 马克斯 | 现在是什么季节？ | 周期位置判断 |
| 索罗斯 | 市场错在哪？ | 反身性拐点识别 |
| 西蒙斯 | 概率怎么说？ | 期望值+置信度 |

## 适配1万本金

- **第一原则**：不亏本金。安全边际是底线，不是参考线
- **执行约束**：单笔亏损>8%无条件止损，总亏损>300暂停一周
- **资金分配**：3000余额宝 + 3000ETF波段 + 3000可转债 + 1000国债应急
- **当前状态**：全市场高估，80%+现金等待7-8月窗口

## 决策流程

1. **格雷厄姆过滤**：安全边际够不够？不够→不碰
2. **芒格排除**：有没有死穴？有→不碰
3. **马克斯定位**：周期在哪？高估区→等待，低估区→考虑
4. **索罗斯判断**：市场有没有反身性扭曲？有→逆向机会
5. **西蒙斯量化**：期望值正吗？概率多大？→最终决策

---

> 每一层都是过滤网，不是加分项。五层全过才动手。
