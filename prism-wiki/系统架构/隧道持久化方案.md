## 推荐方案：Named Tunnel + 廉价域名

### 步骤
1. **注册Cloudflare账号**（免费）
2. **购买廉价域名**（推荐：.xyz/.site首年¥10-30）
   - 腾讯云/阿里云/Cloudflare都行
   - 建议域名：`prism-invest.xyz` 或 `prismcapital.site`
3. **域名NS指向Cloudflare**（Cloudflare免费DNS+CDN）
4. **创建Named Tunnel**
   ```bash
   cloudflared tunnel login  # 认证
   cloudflared tunnel create prism  # 创建命名隧道
   cloudflared tunnel route prism prism.prism-invest.xyz  # 绑定域名
   ```
5. **配置持久化**
   ```yaml
   # ~/.cloudflared/config.yml
   tunnel: prism
   credentials-file: /root/.cloudflared/<tunnel-id>.json
   ingress:
     - hostname: prism.prism-invest.xyz
       service: http://localhost:8900
     - service: http_status:404
   ```
6. **启动（URL固定不再变）**
   ```bash
   cloudflared tunnel run prism
   ```

### 优势
- **URL永久固定**：`https://prism.prism-invest.xyz`
- **免费HTTPS**：Cloudflare自动签发证书
- **免费CDN**：全球加速
- **免费DDoS防护**
- **重启不变化**：named tunnel的域名由DNS CNAME指向，不受进程重启影响

### 成本
- 域名：¥10-30/年（.xyz首年最便宜）
- Cloudflare：免费套餐足够
- 总计：不到¥3/月
